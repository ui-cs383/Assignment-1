Tasks are broken into three categories based on the type of game being played. Each game has a series of tasks, and sub-tasks if they inherit from a parent. The game types have a numeric label which shows the total number of tasks for a particular game type. Each task is uniquely identified via a numeric badge, and the badge keeps a running total of the number of tasks in a particular game.

Subsections with blue backgrounds are not considered tasks, but group similar tasks together.
