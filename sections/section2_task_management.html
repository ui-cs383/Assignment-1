Tasks are broken into 3 categories, based on the type of game being played. Each game has a series of tasks, and sub-tasks if they inhereit from a parent. The game types have a numeric label which shows the total number of tasks for a particular game type.

Tasks with a blue background denote a parent task, and can be clicked on for a list of sub-tasks.

  • Star System Game 10
  • Province Game
  • Galactic Game

Combat

Interaction between opposing player units.

  • Character Combat

    Combat between characters.

  • Environ Military Combat

    Plantary combat between militaries.

Movement

Unit movement between points on the game surface.

  • Interplanetary Movement

    Movement between planets.

  • Planetary Movement

    Movement between enivorns.

Missions

Enivorn-specific actions based on cards.

  • Mission Assignment

    Assignment of missions to characters.

  • Mission Actions

    Actions user must take to complete mission.

  • Search

    Searching for enemy units.
